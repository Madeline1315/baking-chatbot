# 🍰 Baking Assistant Chatbot

A conversational AI chatbot built with Streamlit and the Google Gemini API that acts as an encouraging baking assistant for bakers of all skill levels.

## Features
- Real-time chat interface built with Streamlit
- Conversation memory that persists across messages
- Custom system prompt that keeps responses focused on baking
- Secure API key management with .env file

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- python-dotenv

## Setup
1. Clone the repo
2. Install dependencies: `pip install streamlit google-genai python-dotenv`
3. Create a `.env` file with your Google AI Studio API key:

4. Run the app: `streamlit run baking_chat.py`
